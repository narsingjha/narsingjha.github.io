In this project, I solved a 2-D heat equation using Gauss-Siedel, Successive Over-Relaxation and the Conjugate Gradient method. Later I parallelized the conjugate gradient method using Message Passing Interface (MPI). I had implemented the Compressed Sparse Row format to store the sparse pentadiagonal matrix. The code was written in C. This solver could solve for upto 9 million x 9 million matrices. This was done during the summer internship after the junior year.
