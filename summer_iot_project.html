This project was done during the summer school on IoT. This was done in a team of 5, all the students were from IISc and IIT's. We developed an application that would detect abnormal events during a car journey using the data obtained from the accelerometer. We also had a camera fit on the raspberry pi. We would detect sharp turns, bumps and possible accidents. In case of possible accidents, we would immediately start video recording and send a message along with the video clip to the subscribed parties, for confirmation of accident and providing help, if needed.
